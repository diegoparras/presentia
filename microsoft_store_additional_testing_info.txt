Presenton - Microsoft Store Certification Testing Notes

Purpose
Presenton is a desktop AI presentation generator. It lets users configure their own text and image providers, generate presentations from prompts or documents, edit slides, and export PPTX or PDF files.

Credentials
Please use the credentials supplied in the Microsoft Partner Center "Credentials" section.

Suggested credential names:
1. OPENAI_API_KEY
   Use for Text Provider: OpenAI.
2. PEXELS_API_KEY or PIXABAY_API_KEY
   Use for Image Provider: Pexels or Pixabay, depending on the key supplied.

Test Requirements
1. Install and launch the submitted Presenton desktop app.
2. Internet access is required for OpenAI text generation and Pexels/Pixabay image search.
3. No separate Presenton account is required for this BYOK test path.

API Key Setup

There are two ways to configure providers:

Way 1: First-run onboarding
1. Open the app for the first time.
2. Follow the onboarding flow shown on launch.
3. Select OpenAI as the text provider.
4. Paste the supplied OPENAI_API_KEY.
5. Click Check models if shown, then select any available OpenAI model.
6. Enable image generation/search.
7. Select the matching image provider:
   - Pexels, if the supplied key is a PEXELS_API_KEY.
   - Pixabay, if the supplied key is a PIXABAY_API_KEY.
8. Paste the supplied image API key.
9. Save/continue until the main app opens.

Way 2: Change providers later
1. From the app dashboard, open Settings.
2. Use Text Provider to change OpenAI settings or update OPENAI_API_KEY.
3. Use Image Provider to change Pexels/Pixabay settings or update the image API key.
4. Click Save Configuration.
5. Confirm the settings saved message appears.

Presentation Generation Test
1. Go to Create, Upload, or New Presentation.
2. Enter this prompt:
   Create a 5 slide business presentation about the benefits of AI-powered presentation automation for small teams.
3. Select 5 slides and English.
4. Click Next.
5. Wait for the outline to finish.
6. Open Select Template and choose any template.
7. Click Generate Presentation.
8. Wait until the presentation editor opens.

Optional Document Test
1. Start a new presentation.
2. Upload a supported file such as PDF, DOCX, PPTX, TXT, CSV, XLSX, JPG, or PNG.
3. Add a short prompt, select English, then continue through outline, template selection, and generation.

Editor Test
1. Select any generated slide.
2. Edit a small piece of text.
3. If an image is present, try image replacement or stock image search.
4. Confirm the slide updates without crashing.

Export Test
1. In the editor, click Export.
2. Select PPTX and confirm a PowerPoint file is created.
3. Open the PPTX and confirm slides are readable/editable.
4. Return to the app, click Export again.
5. Select PDF and confirm a PDF file is created.
6. Open the PDF and confirm slides render correctly.

Expected Result
The app should allow API key setup through onboarding or Settings, generate a presentation, allow basic slide edits, and export successfully to PPTX and PDF.

Troubleshooting
1. Generation time depends on internet speed and provider response time.
2. If image search fails, confirm the selected image provider matches the supplied key type.
3. If OpenAI model loading fails, confirm the OPENAI_API_KEY is valid and click Check models again.
4. Provider configuration is stored locally and can be changed from Settings.
